== Description ==

Knowledge Google par JM Créa permet d’ajouter vos liens vers les réseaux sociaux directement dans les résultats Google.

Le plugin est entièrement paramétrable et permet de rentrer les URL vers les réseaux sociaux, et de renseigner votre logo et numéro de téléphone pour qu’ils apparaissent eux aussi dans les résultats.

Le code javascript se place automatiquement entre les balises head.

== Installation ==

1. Uploadez `knowledge—google-par-jm-crea` dans le dossier `/wp-content/plugins/` de votre site
1. Activez ’Knowledge Google par JM Créa’ depuis le menu « Extensions » de Wordpress
1. Rendez-vous ensuite dans l’onglet ‘Outils -> Knowledge Google’
